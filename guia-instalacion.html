1
Instalar Python
Versión 3.11 o superior · Tiempo estimado: 5 minutos
ℹ️ Antes de instalar, verificá si ya tenés Python. Abrí el buscador de Windows, escribí cmd, y en la ventana negra que abre escribí python --version. Si aparece un número como Python 3.11.x, ya tenés Python y podés saltar al Paso 2.
1

Ir a python.org

Abrí tu navegador y entrá a python.org/downloads. Vas a ver un botón amarillo grande que dice "Download Python 3.x.x". Hacé clic ahí.

2

Ejecutar el instalador

Abrí el archivo descargado (python-3.x.x-amd64.exe). Antes de hacer clic en "Install Now", revisá que las dos opciones de abajo estén marcadas.

🚨 MUY IMPORTANTE: Marcá la casilla "Add python.exe to PATH" antes de instalar. Sin eso, Python no va a funcionar desde el terminal. Es la casilla que está abajo de todo en el instalador.
3

Completar la instalación

Hacé clic en "Install Now" y esperá que termine. Cuando aparezca el mensaje "Setup was successful", la instalación terminó.

4

Verificar en el terminal

Cerrá cualquier ventana de cmd que tengas abierta, abrí una nueva (buscá "cmd" en Windows) y escribí:

Command Prompt
python --version
Python 3.11.x ← tiene que aparecer algo así

pip --version
pip 23.x.x from ... ← y esto también
⚠️ Si dice 'python' no se reconoce, probá con py --version. Si tampoco funciona, avisá a los tutores — puede que el PATH no se configuró bien.
2
Instalar VS Code
Visual Studio Code — editor de código gratuito · Tiempo estimado: 3 minutos
ℹ️ VS Code es el editor que vamos a usar para escribir código. Es gratuito, muy popular y tiene autocompletado inteligente. Si ya tenés instalado VS Code, pasá directo al Paso 3.
1

Descargar VS Code

Entrá a code.visualstudio.com. Hacé clic en el botón azul de descarga — automáticamente detecta tu sistema operativo.

2

Instalar

Ejecutá el archivo descargado (VSCodeUserSetup-x64-x.x.x.exe). En las opciones, marcá:

  • Agregar "Abrir con Code" al menú contextual de archivos
  • Agregar "Abrir con Code" al menú contextual de carpetas
  • Agregar al PATH (recomendado)

Después hacé clic en "Instalar" y esperá que termine.

3

Abrir VS Code

Abrí VS Code desde el acceso directo en el escritorio o buscándolo en el menú de Windows. Vas a ver una pantalla de bienvenida azul/oscura.

3
Instalar la extensión de Python
Agrega autocompletado, resaltado de sintaxis y herramientas para Python · 2 minutos
1

Abrir el panel de extensiones

En VS Code, hacé clic en el ícono de cuadraditos del menú lateral izquierdo (parece cuatro cuadrados), o usá el atajo Ctrl + Shift + X.

2

Buscar "Python"

En el buscador de extensiones escribí Python. El primer resultado debería ser "Python" publicado por Microsoft con millones de instalaciones.

3

Instalar

Hacé clic en el botón azul "Install" de esa extensión. Se instala en segundos.

💡 También instalá "Jinja" (buscala igual) — agrega resaltado de colores para las plantillas HTML que vamos a usar en el taller.
4
Crear la carpeta del proyecto y el entorno virtual
Hacemos esto en el terminal integrado de VS Code · 5 minutos
📁 El entorno virtual (venv) es una "caja" separada para tu proyecto. Las librerías que instalés acá no afectan al resto de tu computadora. Siempre hay que activarlo antes de trabajar.
1

Crear la carpeta del proyecto

En VS Code, andá a Archivo → Abrir carpeta. Navegá a donde querés guardar el proyecto (por ejemplo en tu Escritorio o en Documentos) y creá una carpeta nueva llamada taller-flask. Después seleccionala.

2

Abrir el terminal de VS Code

Usá el atajo Ctrl + ` (la tecla con la tilde que está antes del 1) o andá a Terminal → Nuevo Terminal. Se abre un panel abajo del editor — ya está apuntando a tu carpeta del proyecto.

3

Crear y activar el entorno virtual

Copiá y pegá estos comandos uno por uno en el terminal. Esperá que cada uno termine antes de pegar el siguiente:

Terminal en VS Code
# 1. Crear el entorno virtual
python -m venv venv

# 2. Activarlo (Windows CMD/PowerShell)
venv\Scripts\activate

(venv) C:\Users\...\taller-flask> ← así se ve cuando está activo
🚨 Si el terminal dice "la ejecución de scripts está deshabilitada", ejecutá este comando primero y después volvé a activar el venv:
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
5
Instalar Flask y SQLAlchemy
Con el entorno virtual activo · 1 minuto
⚠️ Verificá que el terminal muestra (venv) al inicio de la línea antes de instalar. Si no lo ves, el entorno virtual no está activo — volvé al Paso 4.
Terminal — con (venv) activo
# Instalar Flask y Flask-SQLAlchemy
pip install flask flask-sqlalchemy

Collecting flask...
Collecting flask-sqlalchemy...
...
Successfully installed Flask-3.x.x Jinja2-3.x ... ← tiene que aparecer esto
💡 Si la instalación tarda más de 2 minutos, verificá tu conexión a internet. Si da algún error, avisá a los tutores — no intentes resolver errores de pip solo.
6
Verificación final — ¡Todo listo!
Probamos que Flask funciona de principio a fin · 3 minutos
1

Crear el archivo de prueba

En VS Code, creá un archivo nuevo (Archivo → Nuevo archivo) y guardalo como test.py en la carpeta taller-flask. Copiá este código:

test.py
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hola():
return '<h1>¡Flask está funcionando! 🎉</h1>'

if __name__ == '__main__':
app.run(debug=True)
2

Correrlo

En el terminal (con el venv activo), escribí:

Terminal
python test.py

* Running on http://127.0.0.1:5000
* Debug mode: on
* Restarting with stat
3

Ver el resultado en el navegador

Abrí tu navegador (Chrome, Edge, Firefox) y entrá a http://localhost:5000.

🎉

Si ves "¡Flask está funcionando!" en el navegador, tu entorno está perfecto.

Podés cerrar el servidor con Ctrl + C en el terminal. Nos vemos en el taller.

Si algo no funciona, no te preocupes — avisá a los tutores antes del taller o al inicio. Vamos a resolverlo juntos. Lo único que necesitás para el taller es tener Python instalado; el resto lo hacemos en el momento.

7
Instalar Git y bajar el repo del taller
Solo si no tenes Git · 5 minutos
ℹ️ Git sirve para descargar el codigo del taller y luego actualizarlo. Si ya tenes Git instalado, salta al paso de clonar.
1

Instalar Git (Windows)

Entrá a git-scm.com/downloads y descargá Git. Ejecutá el instalador con las opciones por defecto.

2

Verificar Git

Abrí una terminal nueva y escribí:

Terminal
git --version
git version 2.x.x
3

Clonar el repo del taller

En la carpeta donde queres guardar el proyecto, ejecuta:

Terminal
git clone URL_DEL_REPO
cd taller-flask-repo
💡 Reemplaza URL_DEL_REPO por el link que compartan los tutores.
4

Actualizar el repo (pull)

Si el repo ya esta clonado y queres traer cambios nuevos:

Terminal
cd taller-flask-repo
git pull
⚠️ Si git pull muestra errores, no borres nada. Avisá a los tutores.
📋
Resumen — Comandos que vas a usar todos los días
Guardá esto para referencia rápida

Al empezar a trabajar:

# 1. Entrar a la carpeta
cd taller-flask
# 2. Activar el entorno
venv\Scripts\activate
# 3. Correr la app
python app.py

Señales de que todo está bien:

  • Terminal muestra (venv)
  • Terminal muestra Running on http://...
  • El navegador muestra tu app
  • Al cambiar código y guardar, el servidor se recarga solo
